# 텍스트 기반 MMO RPG 개발 계획서

## 프로젝트 개요
- **목표**: 이미지 없는 완전 텍스트 기반 MMO RPG
- **기술스택**: Next.js, TypeScript, React
- **데이터 저장**: JSON 파일 기반
- **관리**: 관리자 대시보드에서 모든 기능 제어
- **성장시스템**: 무한 스킬트리, 스탯, 레벨 시스템
